About This Tool

Chrome extensions require icons in multiple sizes for different contexts. This tool helps you generate all the required icon sizes from a single image, saving you time and ensuring consistency across all icon sizes.

How to Use

  1. Upload a square image (PNG or SVG recommended)
  2. Preview the generated icons in different sizes
  3. Download a ZIP file containing all the required icon sizes

Required Icon Sizes

Chrome Web Store requires the following icon sizes for extensions:

  • 16x16: Favicon for extension
  • 32x32: Windows computers often require this size
  • 48x48: Displayed in the Chrome Web Store
  • 128x128: Displayed in the Chrome Web Store and during installation

Privacy Note

All image processing is done locally in your browser. Your images are never uploaded to our servers.
