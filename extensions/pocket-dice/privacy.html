Privacy Policy

Pocket Dice Chrome Extension

Last Updated: March 26, 2025

Introduction

Thank you for using the Pocket Dice Chrome extension. This Privacy Policy explains what information the extension collects, how it uses that information, and how your data is protected.

Information Collection and Use

The Pocket Dice Chrome Extension is designed with privacy in mind:

  • No Personal Data Collection: The extension does not collect, store, transmit, or share any personal information from users.
  • Local Storage Only: All game logic, scores, and states are processed and stored locally within your browser session for the duration the extension popup is open and are discarded when the popup is closed.
  • No Analytics or Tracking: The extension does not use any analytics, tracking tools, or cookies.
  • No External Services: The extension operates entirely offline and does not communicate with external servers.

Permissions Required

The extension requests the following permissions in the Chrome browser for the sole purpose of providing its core functionality:

  • Scripting: To run the JavaScript necessary for the game's functionality within its own popup window.
  • Active Tab: Included as part of standard Manifest V3 practices, but not actively used to interact with web page content.

Data Security

Since all data is stored locally on your device and only for the duration of your session, the security of your data is maintained by your browser's built-in security features. We do not have access to this data.

Changes to This Privacy Policy

We may update our Privacy Policy from time to time. Any changes will be reflected in the extension's description on the Chrome Web Store and/or in the extension itself. The latest version will always be available on this page.

Contact Us

If you have any questions about this Privacy Policy, please reach out to us at hello@startupmachine.co.

By using the Pocket Dice extension, you agree to the terms outlined in this Privacy Policy.
