Privacy Policy for Nyan Cat Progress Bar Chrome Extension

Last Updated: February 28, 2024

Overview

Nyan Cat Progress Bar is committed to protecting your privacy. This privacy policy explains our data collection, use, and protection practices.

No Personal Data Collection

We want to be clear: The Nyan Cat Progress Bar Chrome Extension does not collect any personal user data.

This extension is designed to simply replace the default page loading progress bar with the Nyan Cat animation. It does not track your browsing history, collect any information about the websites you visit, or gather any personally identifiable information.

Permissions

The Nyan Cat Progress Bar extension requires the following Chrome permissions:

  • "activeTab": To access current webpage information in order to inject the Nyan Cat progress bar.
  • "scripting": To inject JavaScript and CSS code into webpages to display the Nyan Cat progress bar.

These permissions are solely used for the functionality of the extension and are not used to collect any user data.

Data Storage and Transmission

What We Don't Collect:

  • Personal information
  • Complete browsing history
  • Login credentials
  • Payment information
  • Individual user identifiers
  • Content of web pages you visit

Data Storage:

  • No citation data is stored as the extension does not create or manage citations.
  • No user preferences are stored locally.

Data Transmission:

  • No data is transmitted to our servers.
  • No data synchronization between devices.
  • No third-party access to any user data.

User Control

You have complete control because no data is collected:

  • No citation history to clear (as none is collected).
  • No preferences to reset (as none are stored).
  • No account required.
  • No data lock-in.

Updates

We may update this privacy policy from time to time. We will notify users of any significant changes through the extension's Chrome Web Store page or our website.

Contact Us

If you have any questions about this privacy policy, please contact us at: hello@startupmachine.co

By using the Nyan Cat Progress Bar Chrome Extension, you acknowledge that you have read and understood this privacy policy.
