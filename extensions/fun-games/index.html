Fun Games

A collection of six casual games to play right in your browser

About Fun Games

Fun Games is a Chrome extension that brings six classic mini-games directly to your browser. Perfect for quick breaks, waiting times, or just when you need a moment of entertainment without leaving your browser.

All games run completely offline and require no internet connection once installed. The extension is lightweight, fast, and provides hours of casual entertainment.

Games Included

  • Rock Paper Scissors: Play the classic game against the computer. Choose between rock, paper, and scissors and see who wins!
  • Number Guessing Game: Can you guess the secret number? The computer will pick a random number between 1 and 100, and you need to guess it with hints.
  • Hangman: The classic word guessing game. Guess the word letter by letter before the stick figure is complete.
  • Tic-tac-toe: Play against the computer in this strategic game. You're X, the computer is O.
  • Word Scramble: Unscramble letters to form words. The game provides a hint for each scrambled word.
  • Snake: Control a snake to eat food and grow without hitting walls or yourself.

Key Features

  • Six different casual games in one compact extension
  • Intuitive controls with keyboard support for applicable games
  • Sound effects for all games with a global mute option
  • Score tracking with best score saved
  • No internet connection required after installation
  • No ads or in-app purchases - completely free to use

How to Use

  1. Install the extension from the Chrome Web Store
  2. Click on the Fun Games icon in your Chrome toolbar
  3. Select any game from the menu to start playing
  4. Click the "Back" button to return to the games menu at any time
